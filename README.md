# The Future of the Catskill Interpretive Center

A repository for the website http://future.cic.design

Install yarn on your machine: [https://yarnpkg.com/en/docs/install](https://yarnpkg.com/en/docs/install)

- Install packages
```
yarn
```

- Build the site
```
yarn develop
```

- Run dev server
```
yarn build
```
